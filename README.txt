
Code for the paper: 

Noisy-Defense Variational Auto-Encoder (ND-VAE): An Adversarial Defense Framework to Eliminate Adversarial Attacks

https://ieeexplore.ieee.org/document/10387596

Run Instructions:

1. Change to src directory
2. set parameters in config.py
3. comment/uncomment your desired tests in test_adversarial.py (or leave as is to run everything)
4. run with "python test_adversarial.py"

NOTE: If you are running out of GPU memory, try reducing batch size and NVAE parameters.
